All about Accessibility in HTML

Web accessibility means that websites,tools and technologies are designed and developed so that people with disabilities can use them. More specially ,people can:

  • Perceive,understand,navigate,and interact with the web
  • contribute to the web
Web accessibility encompasses all disabilities that affect access to the web,including
  • auditory
  • cogative
  • neurological
  • physical
  • speech
  • visual

ways to make html accessible

web content can be made accessible just making sure the correct HTML elements are used for the correct purpose. Some major ways to make HTML accessible are as follows:

  • Semantic HTML
  • Good Alternative texts
  • Proper use of Headings
  • HTML lang Attribute
  • Descriptive link text

Features of accessibily in different user groups

  • Visibility & visual clarity
  • Clear,simple language
  • accessible CTAs
  • accessible forms
  • Subtitles,Captions & transcripts for video

    • Benefits

    It is safe to say the accessibility benefits all members of society including people with disabilities.Improving accessibility brings about increased quality of life,creates more independence and better social intergation.It also leads to better health and can result in cost saving in no. of areas.
